# MiraFashion - Vue SPA E-Commerce Website

MiraFashion is a single page application developed using Vue 3, TypeScript, Vite, and Tailwind CSS. It is a fashion-themed e-commerce website that allows users to browse products, filter by categories, search products, view product details, add products to the cart, and complete a demo checkout process.

## Project Features

- Responsive home page for mobile, tablet, and desktop screens
- Fashion-themed hero section with local background images
- Product data loaded from DummyJSON
- Product categories:
  - Clothes
  - Footwear
  - Others
- Product search functionality
- Product details modal without opening a new page
- Add to cart functionality
- Quantity selection in product details
- Size selection for clothing products
- Cart page with subtotal, shipping, and total calculation
- Checkout modal with delivery information and payment method
- Credit/Debit card and Cash on Delivery options
- Success message popup after payment confirmation
- Login page with demo pre-filled credentials
- Dark mode support
- Floating cart button with cart item count
- Working contact popup in footer

## Technologies Used

- Vue 3
- TypeScript
- Vite
- Tailwind CSS
- Pinia
- Vue Router
- DummyJSON API
